# Ma Constellation Docs

Prototype de site interactif pour le systeme Point Zero, cree en aout 2025.

## Contenu

| Fichier | Description |
|---------|-------------|
| `index.html` | Page d'accueil interactive — navigation narrative avec choix, teasers, liens vers les sous-pages |
| `docs/glossaire.html` | Version HTML du glossaire |
| `docs/aquette Interactive - Échos de la Constellation Onirique (Mise à Jour)` | Maquette d'interface pour le volet onirique |
| `SOM STR const` | Sommaire structure de "La Constellation du Point Zero" — plan complet en 5 parties |

## Le sommaire

Le fichier `SOM STR const` contient le plan de l'ouvrage de synthese (Livre 7 de la serie Constellation) :

- **Partie I** — Les Fondements Ontologiques (PZ, Intelligence Idiopathique, Geometrie Qualitative)
- **Partie II** — Les Mecaniques de l'Existence (Torsion Vectorielle, Ether Fluide, Machine Temporelle)
- **Partie III** — Application Concrete (Psychologie Gestationnaire, Conscience, Organisation Systemique)
- **Partie IV** — Technologie Qualitative (INTemple, MetaConjugaison)
- **Partie V** — Perspectives et Formalisations

## Statut

Prototype exploratoire. Utilise Tailwind CSS via CDN. Les pages sont fonctionnelles mais le contenu est en grande partie a remplir (placeholders et structures).

Ce depot date d'avant la reorganisation du corpus en CLR SYSTM (mars 2026).
